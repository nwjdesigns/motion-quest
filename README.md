# Motion Quest

A persistent, structured home for motion design learning and production knowledge across **Cavalry**, **After Effects**, **live action**, and **3D** (Cinema 4D / Blender) — built under [East End Switchyard](./product.md).

**Current spike**: Cavalry proficiency.

Every session produces something. Speed drills 15–30 min. Learn by building, not watching.

For deeper project context (studio model, why this exists, constraints) see [`product.md`](./product.md).

## Folders

| Folder | Purpose |
|---|---|
| `cavalry/` | Cavalry-specific exercises and node setups |
| `after-effects/` | AE techniques, expressions, templates |
| `live-action/` | Shooting, editing, production notes |
| `3d/` | Cinema 4D / Blender experiments |
| `exercises/` | Daily drills (any tool) |
| `projects/` | Longer builds, spec work scenes |
| `notes/` | Learnings, cross-tool translation notes |
| `references/` | Screenshots, links, inspiration |
